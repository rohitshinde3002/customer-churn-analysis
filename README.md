## 📘 Customer Churn Analysis

### 📌 Project Overview

This project analyzes telecom customer data to predict churn using Logistic Regression.

---

### 🛠 Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

### 📊 Steps Performed

1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis (EDA)
4. Data Encoding
5. Feature Scaling
6. Model Building (Logistic Regression)
7. Model Evaluation

---

### 📈 Model Accuracy

The Logistic Regression model achieved **88% accuracy**.

---

### 📌 Key Insights

* Customers with more inactive days are likely to churn.
* Higher support calls increase churn probability.
* Longer subscription reduces churn.
